# Vue 3 example with MindAR 

This is an example project of using [MindAR](https://github.com/hiukim/mind-ar-js) in Vue 3 

# Screenshot

# It demonstrates:

1. how to import MindAR as a npm package
2. how to create a component for MindAR in Vue 3

# To run
```
yarn install
```

```
yarn serve
```
